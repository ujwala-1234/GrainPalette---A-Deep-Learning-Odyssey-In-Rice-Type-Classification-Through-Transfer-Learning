Rice Type Classification Model using CNN

This model is built using a Convolutional Neural Network and trained on labeled images of rice. It takes an image of a rice grain as input and predicts its type.

Accuracy of the Model

The model gives correct predictions 97 times out of 100.

Different Types of Rice

This model can classify 5 different rice types:

  • Arborio
  • Basmati
  • Ipsala
  • Jasmine
  • Karacadag

Dataset Used

The dataset of labeled rice images is obtained from Kaggle.

Technical Architecture

The model is trained using a Convolutional Neural Network.
